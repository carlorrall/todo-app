# Vanilla JS Todo App
A simple Todo application built with HTML, CSS, and JavaScript.

Live Demo: https://carlorrall.github.io/todo-app/

## Features
- Add todos
- Edit todos
- Delete todos
- Save todos using LocalStorage
- Press Enter to save
- Shift + Enter for new line

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage API

## How to Run
Open `index.html` in your browser.
